# 📰 Ossiqn Haber Yazılımı

Performans, güvenlik ve sadelik odaklı, saf PHP ile geliştirilmiş modern haber scripti. Gereksiz şişirilmiş kodlardan arındırılmış, doğrudan sonuca odaklanan profesyonel bir altyapı.

---

## ✨ Öne Çıkan Özellikler

* 🚀 **Yüksek Performans:** Optimize edilmiş veritabanı sorguları ve hafif kod yapısı ile ultra hızlı sayfa yükleme süreleri.
* 🛡️ **Güvenlik Odaklı:** SQL Injection ve XSS saldırılarına karşı sanitize edilmiş veri girişleri.
* 📱 **Tam Mobil Uyumluluk:** Her cihazda kusursuz görünen responsive arayüz.
* 💎 **Temiz Kod Mimarisi:** Karmaşadan uzak, mantıksal olarak hizalı ve bakımı kolay kod dizilimi.
* 🔍 **SEO Dostu:** Arama motorları için optimize edilmiş link yapısı ve meta yönetimi.

---

## 🛠️ Teknik Detaylar

* **Dil:** PHP 8.x
* **Veritabanı:** MySQL / MariaDB
* **Tasarım:** Modern CSS / HTML5
* **Mimari:** Fonksiyonel & Modüler Yapı

---

## ⚙️ Kurulum (Installation)

1.  **Dosyaları Yükleyin:** Tüm dosyaları sunucunuzun ana dizinine yükleyin.
2.  **Veritabanı Oluşturun:** `sql` klasörü içerisindeki `.sql` dosyasını veritabanınıza içe aktarın.
3.  **Yapılandırma:** `config.php` (veya veritabanı bağlantı dosyanız) içerisinden DB bilgilerinizi güncelleyin.
4.  **Hazır!** Siteniz anında yayına girmeye hazır.

---

## 🔒 Güvenlik Notları

Bu script geliştirilirken "sıfır güvenlik açığı" prensibiyle hareket edilmiştir. Tüm değişkenler kontrol edilmeden işleme alınmaz. Sistem içerisinde geliştirici notu veya yorum satırı barındırmayan, doğrudan üretim aşamasına (production) hazır kod yapısı mevcuttur.
